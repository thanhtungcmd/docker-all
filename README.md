# Docker service

Uncomment from file docker-compose.yml to start service

1. Start docker
```
docker-compose up -d
```
2. Stop docker
```
docker-compose down
```

Note

> PHP need file 
> - docker/php/instantclient-basic-linux.x64-12.1.0.2.0.zip
> - docker/php/instantclient-sdk-linux.x64-12.1.0.2.0.zip
